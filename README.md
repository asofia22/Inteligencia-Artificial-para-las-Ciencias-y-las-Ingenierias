Nombre: Ana Sofía Murillo Hernández
Correo institucional: asofia.murillo@udea.edu.co

Descripción

Este repositorio contiene la solución de la FASE 1 del ejercicio de Programación con LLMs.

Se incluyen:

4 preguntas diseñadas relacionadas con procesamiento de datos y Machine Learning.
4 generadores de casos de uso aleatorios para validar automáticamente las soluciones.
Cada generador produce:

Un input (diccionario con argumentos de la función).
Un output esperado (resultado correcto según la lógica del problema).
